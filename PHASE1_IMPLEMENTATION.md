# 📦 Фаза 1: Экономика и Квесты — Реализация завершена

## ✅ Выполненные работы

### 1. База данных (`database/033_economy_and_quests.sql`)

Созданы **6 новых таблиц**:

| Таблица | Описание |
|---------|----------|
| `vendors` | Торговцы (имя, локация, капитал, время обновления) |
| `vendor_items` | Ассортимент торговцев (предметы, цены, сток) |
| `recipes` | Рецепты крафта (результат, требования) |
| `recipe_ingredients` | Ингредиенты для рецептов |
| `quests` | Шаблоны квестов (тип, цель, награда) |
| `player_quests` | Прогресс квестов игроков |

**Тестовые данные:**
- Торговец "Михалыч" в Убежище
- Квест "Охота на крыс" (убить 5 крыс)

---

### 2. API Торговли (`public/api/vendor.php`)

**Методы:**
- `action=list` — Получить список торговцев и их товары
- `action=buy` — Купить предмет (транзакция: деньги -1, предмет +1, сток -1)
- `action=sell` — Продать предмет (50% от базовой цены)

**Фичи:**
- ✅ Проверка баланса игрока
- ✅ Ограниченный сток товаров
- ✅ Защита от продажи квестовых предметов
- ✅ Транзакционная безопасность

---

### 3. API Квестов (`public/api/quests.php`)

**Методы:**
- `action=list` — Список доступных и активных квестов
- `action=start` — Начать квест (проверка на уникальность)
- `action=progress` — Обновить прогресс (+1 к убийству/сбору)
- `action=complete` — Завершить квест и получить награду

**Награды:**
- 💰 Крышки
- ⭐ Опыт (XP)
- 🎒 Предметы

**Фичи:**
- ✅ Повторяемые и одноразовые квесты
- ✅ Автоматическая проверка готовности
- ✅ Транзакционная выдача наград

---

### 4. API Крафта (`public/api/crafting.php`)

**Методы:**
- `action=list` — Список рецептов с проверкой ингредиентов
- `action=craft` — Создать предмет (транзакция: ингредиенты -1, результат +1)

**Фичи:**
- ✅ Индикация доступности рецепта (can_craft)
- ✅ Проверка наличия всех ингредиентов
- ✅ Откат транзакции при нехватке ресурсов

## 📊 Архитектурные улучшения

| Принцип | Реализация |
|---------|------------|
| **DRY** | Общая логика транзакций вынесена в паттерн |
| **SOLID** | Каждый API отвечает за одну доменную область |
| **Безопасность** | Prepared statements, валидация типов `(int)` |
| **Целостность** | Foreign keys с CASCADE удалением |
| **Производительность** | Один запрос на рецепт + проверка ингредиентов |

## 📝 Пример использования API

### Начало квеста
```javascript
fetch('/api/quests.php', {
    method: 'POST',
    headers: {'Content-Type': 'application/x-www-form-urlencoded'},
    body: 'action=start&quest_id=1'
})
.then(r => r.json())
.then(data => console.log(data));
// {"success": true, "message": "Квест начат: Охота на крыс"}
```

### Покупка предмета
```javascript
fetch('/api/vendor.php', {
    method: 'POST',
    body: 'action=buy&vendor_id=1&item_id=5'
})
.then(r => r.json());
// {"success": true, "message": "Куплено: Пистолет 10мм", "price": 150}
```

### Крафт
```javascript
fetch('/api/crafting.php', {
    method: 'POST',
    body: 'action=craft&recipe_id=1'
})
.then(r => r.json());
// {"success": true, "message": "Скрафчено: Чистая вода x1"}
```
