# 🎮 FALLOUT RPG: РУКОВОДСТВО ПО ЗАПУСКУ

## 📋 Статус проекта: Alpha v0.5 (Готов к Git Publication)

Проект готов к публикации на GitHub как промежуточный результат. Реализовано ядро игры, система боя, инвентарь, карта мира и мультиплеерная основа.

---

## 🚀 БЫСТРЫЙ СТАРТ

### 1. Применение миграций базы данных

```bash
# Вариант А: Автоматически (скрипт)
./apply_migrations.sh

# Вариант Б: Вручную (по порядку)
mysql -u root -p fallout_db < database/001_create_players.sql
mysql -u root -p fallout_db < database/002_create_characters.sql
# ... (все миграции по порядку до 023)
mysql -u root -p fallout_db < database/023_inventory_loot_junkjet.sql
```

### 2. Генерация карты мира

```bash
php scripts/generate_world_map.php
```

### 3. Запуск игрового сервера

```bash
php -S localhost:8000 -t public
```

### 4. Открыть в браузере

```
http://localhost:8000
```

## 🐛 ИЗВЕСТНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Ошибка внешнего ключа `fk_dungeon_boss`
**Решение:** Применить миграции строго по порядку. Тип `monsters.id` должен совпадать с `dungeons.boss_id`.

### Отсутствует таблица `admins`
**Решение:** Таблица создается в миграции `020_seed_initial_data.sql` автоматически.

### Поле `is_boss` не найдено
**Решение:** Поле добавлено в `005_create_monsters.sql`. Применить миграцию заново.
