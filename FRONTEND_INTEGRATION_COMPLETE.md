## 🎮 Игровой Процесс

### Квесты:
1. Игрок открывает панель 📋
2. Видит доступные квесты со статусом (📝 В процессе, ✅ Завершен, ❌ Провален)
3. Нажимает "▶️ Начать" для старта
4. Выполняет условия (убить монстров, найти предметы)
5. Нажимает "🎁 Завершить" для получения награды (крышки, опыт, предметы)

### Торговля:
1. Игрок открывает панель 💰
2. Видит список торговцев с локацией и количеством крышек
3. Нажимает "💬 Торговать"
4. Открывается двустороннее окно:
   - **Слева**: товары торговца (кнопка "Купить")
   - **Справа**: инвентарь игрока (кнопка "Продать")
5. Цены: покупка по полной, продажа по 50% от базовой стоимости

### Крафт:
1. Игрок открывает панель 🔨
2. Видит рецепты с требованиями
3. Если ингредиентов хватает - кнопка "🔨 Создать" активна (зеленая)
4. Если не хватает - кнопка заблокирована (серая), показано чего не хватает
5. После создания предмет добавляется в инвентарь
