### 2. Интеграция настроек шансов с админкой

**Файл миграции:** `database/032_add_loot_chance_settings.sql`

**Добавлено 14 настроек:**
- `search_loot_base_chance` — базовый шанс лута
- `search_weapon_chance` — шанс оружия (0.4%)
- `search_armor_chance` — шанс брони (0.2%)
- `search_consumable_chance` — шанс расходников (2.0%)
- `search_caps_chance_base` — шанс крышек (15%)
- `search_caps_min/max` — диапазон крышек
- `search_luck_bonus_multiplier` — множитель удачи
- `search_location_quality_multiplier` — множитель качества локации
- `search_pity_timer_threshold` — порог pity timer (50 поисков)
- `search_cooldown_seconds` — кулдаун поиска
- `search_monster_encounter_base` — шанс встречи с монстром
- `search_monster_danger_multiplier` — опасность локаций

**Результат:** Админ может менять шансы в реальном времени через `/admin.php` → "⚙️ Настройки"

#### 🏪 API Торговли (`public/api/vendor.php`)

```
Методы:
├── action=list       → Список торговцев и товаров
├── action=buy        → Покупка (транзакция)
└── action=sell       → Продажа (50% цены)

Фичи:
✅ Проверка баланса
✅ Ограниченный сток
✅ Защита от продажи квестовых предметов
✅ Транзакционная безопасность
```

#### 📜 API Квестов (`public/api/quests.php`)

```
Методы:
├── action=list       → Доступные квесты

Награды:
├── 💰 Крышки
├── ⭐ Опыт (XP)
└── 🎒 Предметы

Фичи:
✅ Повторяемые/одноразовые квесты
```

## 🏆 Достигнутые улучшения

### Архитектурные

## 📞 Рекомендации для команды

1. **Запустить миграцию БД:**
