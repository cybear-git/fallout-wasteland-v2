## 🔌 Backend API

### 1. `/api/factions.php` - Система Фракций

**Методы:**
- `get_status` - Получить статус всех фракций игрока
- `update_reputation` - Изменить репутацию (для квестов/событий)
- `get_effects` - Получить активные бонусы от репутации

**Ранги репутации:**
```
+1000: Легенда
+750:  Герой
+500:  Почетный член → Скидка 20%, уникальные квесты
+250:  Друг → Скидка 10%
+100:  Союзник → Безопасный проход
+50:   Знакомый
-50:   Нейтрал
-100:  Подозрительный
-250:  Враг → Враждебность
-500:  Заклятый враг → Атаки при встрече
-1000: Изгой → Отказ в торговле
```

### 2. `/api/dungeons.php` - Подземелья и Боссы

**Методы:**
- `get_dungeons` - Список доступных подземелий с проверкой уровня
- `enter_dungeon` - Вход в подземелье (бой с боссом или серия боев)
- `complete_dungeon` - Завершение прохождения с наградой

**Особенности:**
- Проверка минимального уровня
- Автоматическое открытие точки ФП при входе
- 10% шанс на уникальный предмет в награду
- Награда: 50-100 крышек + 20-40 XP

### 3. `/api/fast_travel.php` - Быстрое Перемещение

**Методы:**
- `get_points` - Список открытых точек
- `travel` - Телепортация (стоимость: 10 крышек)
- `discover` - Открытие новой точки

**Ограничения:**
- Нельзя телепортироваться во время боя
- Только открытые ранее локации
- Требуется минимум 10 крышек

## 📝 Примеры Использования

### Изменение репутации после квеста:
```javascript
fetch('/api/factions.php', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        action: 'update_reputation',
        faction_id: 1, // Братство Стали
        change: +100,
        reason: 'complete_quest'
    })
});
```

### Вход в подземелье:
```javascript
fetch('/api/dungeons.php', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        action: 'enter_dungeon',
        location_id: 42
    })
});
```

### Телепортация:
```javascript
fetch('/api/fast_travel.php', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        action: 'travel',
        location_id: 15
    })
});
```
